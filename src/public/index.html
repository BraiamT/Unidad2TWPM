NOSOTROS

  • Geolocalización


    Determina la ubicación física de un objeto o una persona con relación a un sistema de coordenadas (mapa) para, posteriormente, acceder a información específica.


    Aplicaciones como Uber, Didi, Rappi o incluso Tinder, se sirven de esta herramienta para tener mayor precisión en la ubicación de sus usuarios.

  • Web Workers


    Un Web Worker es un script que se ejecuta en segundo plano, sin afectar o bloquear el desempeño de nuestra página web.


    Aunque que no todos los browsers soportan los workers, las versiones más recientes si lo hacen y son una excelente alternativa para lidiar con tareas pesadas que tengan que ejecutarse sin bloquear las páginas a la interacción con el usuario.

  • Web Sockets


    WebSockets es una tecnología avanzada que hace posible abrir una sesión de comunicación interactiva entre el navegador del usuario y un servidor.


    Con esta API, se pueden enviar mensajes a un servidor y recibir respuestas controladas por eventos sin tener que consultar al servidor para una respuesta.

  • Google Maps


    Esta API, proporcionada por Google, permite a los usuarios mostrar un mapa de nuestra ubicación en tiempo real, utilizando los datos de geolocalización para una mayor exactitud.


    Se puede aplicar en sitios web que requieran mostrar su ubicación, o en aplicaciones que busquen mostrar la ubicación, como WhatsApp y Facebook.

  • Notificaciones


    Las notificaciones push son mensajes instantáneos enviados por una aplicación o sitio web, en respuesta a un evento o proceso determinado que tiene lugar en la aplicación o página web.


    Un claro ejemplo de estas notificaciones vienen de WhatsApp Web, que muestra una notificación cada que se recibe un mensaje.
